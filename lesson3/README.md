# Казилова Агенты — мультиагентная система контента

Telegram-бот с командой AI-агентов для создания контента клиники доктора Казиловой.

## Как это работает

Ты пишешь задачу → агенты работают по цепочке:

1. **Маркетолог** — ищет актуальные данные в интернете, составляет бриф
2. **Копирайтер** — пишет готовый текст поста на основе брифа
3. **Дизайнер** — создаёт промпт для генерации картинки

## Что нужно для запуска

- Python 3.9+
- Claude Code (авторизованный через подписку)
- Telegram-бот (токен от @BotFather)

## Установка

```bash
# Создать виртуальное окружение
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# Установить зависимости
pip install -r requirements.txt
```

## Настройка

Создай файл `.env` (скопируй из `.env.example`):

```
TELEGRAM_TOKEN=твой_токен_от_BotFather
CLAUDE_MODEL=claude-sonnet-4-6
```

## Запуск

```bash
python bot.py
```

## Использование в личных сообщениях

Напиши боту:
- `сделай пост про липосакцию для Instagram`
- `напиши про блефаропластику`
- `создай контент про абдоминопластику`

## Использование в группе с ветками

1. Создай группу в Telegram
2. Включи Topics: Настройки группы → Темы
3. Добавь бота как администратора (права: управление темами)
4. Напиши `/setup` — бот создаст ветки для каждого агента
5. Пиши задачи в General — агенты отвечают в своих ветках

## Переключение модели

В `.env` измени `CLAUDE_MODEL`:
- `claude-sonnet-4-6` — оптимальный (рекомендуется)
- `claude-haiku-4-5-20251001` — быстрее, экономит лимиты

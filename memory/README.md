# memory/

Дневные заметки агента. Каждый файл — один день.

Формат имени: `YYYY-MM-DD.md`

Агент создаёт файл за сегодня автоматически при первой записи дня.
Читается только за текущий день — прошлые дни не грузятся в контекст.

## Что здесь хранится

- Задачи дня и их статус
- Что сделано
- Проблемы и решения
- Что не закрыто и переходит на завтра

## Что здесь НЕ хранится

Постоянные факты об Анжеле — они идут в `MEMORY.md`.
Справочники и гайды — они идут в `knowledge/`.
